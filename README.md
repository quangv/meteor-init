# Meteor App Starter

__Features__
- kadira:flow-router
- coffeescript
- less
- accounts-ui


```
git clone git@github.com:quangv/meteor-init.git

mv meteor-init my_app_name
cd my_app_name

git remote rm origin
```

## Dev Server
```
  meteor
```

## To Deploy

```
  meteor deploy my_app_name.meteor.com
```
